# Hotkey Reference
⇧ = Shift
^ = Control
⌥ = Option
⌘ = Command

## Spectacle
- ⌥⌘F = fullscreen current window
- ⌥⌘← = snap window to left
- ⌥⌘→ = snap window to right
- ⌥⌘↑ = snap window to top
- ⌥⌘↓ = snap window to bottom
- ⌥⌘; = snap window to top left corner
- ⌥⌘' = snap window to top right corner
- ⌥⌘. = snap window to bottom left corner
- ⌥⌘/ = snap window to bottom right corner

## Git aliases
- co = checkout
- ci = commit
- st = status
- br = branch
